# Assets Folder

## Required Files

### Board.png
- **Purpose**: Stats board background image for the landing page hero section
- **Location**: Place the Board.png file in this directory (`client/public/assets/Board.png`)
- **Description**: A hanging wooden scroll/board with parchment texture and three icon sections (people, swords, coin)
- **Usage**: Referenced in LandingPage.jsx as `/assets/Board.png`
- **Dimensions**: Approximately 280px width, maintains aspect ratio

## How to Add

1. Save the Board.png image file to this directory
2. The landing page will automatically load it from `/assets/Board.png`
3. Stats text will overlay on top of the board image sections
